# chok-v2-deploy

chok-v2 서비스의 통합 실행과 배포 설정을 관리하는 저장소입니다.

서비스 구현 코드는 각 서비스 저장소에서 관리하고, 이 저장소에는 공용 MySQL과 서비스 이미지의 실행 구성, 배포 및 smoke 검증 스크립트만 둡니다.

## 대상 서비스

- `backend-spring`
- `api-fastapi`
- `py-sdk` (후속 true E2E 단계)

프론트엔드는 Vercel 배포를 기준으로 하며 이 저장소의 컨테이너 구성에는 포함하지 않습니다.

## 현재 상태

저장소 초기 설정 단계입니다. Compose와 DB 초기화, 고정 번들 smoke 구성은 작업 브랜치에서 추가합니다.
